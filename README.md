# Recipedia
**Recipedia** is a full-stack recipe recommendation website that helps users discover meals they can cook based on the ingredients they already have at home.

The app allows users to log in, manage their personal ingredient inventory, and receive recipe suggestions based on full or partial matches. Recipes are fetched via API integrations with **Spoonacular**, and optionally generated using **OpenAI** for smarter suggestions.

## Features
- 🔐 User authentication and session management
- 🧺 Add, remove, and categorize ingredients
- 📦 Save ingredients to personal profile
- 🍽️ Get recipe suggestions:
  - Full match – you have all required ingredients
  - Partial match – based on most matching ingredients
- 🌐 Recipe data from Spoonacular API and/or OpenAI
- 💾 Save favorite recipes
- 🎯 Clean and responsive UI
