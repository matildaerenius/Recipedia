# Recipedia
**Recipedia** is a full-stack recipe recommendation website that helps users discover meals they can cook based on the ingredients they already have at home.

Built with **Java**, **Spring Boot**, **MySQL**, and **React**, the app allows users to log in, save their personal ingredient inventory, and receive recipe suggestions through API or AI integrations.
